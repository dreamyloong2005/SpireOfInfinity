## 📖 简介 (About)

**无尽尖塔 (Spire of Infinity)**
在《杀戮尖塔2》（Slay The Spire 2）中游玩来自《无尽争霸》（War Of Infinity）的特色英雄，体验背景故事。

## ✨ 主要特性 (Features)

* **全新角色、卡牌、遗物等**：在STS2中体验“黑暗守望者”，这是一位来自异次元的神祗，因为穿越到尖塔宇宙而失去了大部分力量，攀爬高塔，找回你的神力并了解这个世界的真相。
* **高度兼容**：完全兼容多人模式。

## 🛠️ 安装指南 (Installation)

1. 确保你已经安装了前置模组（BaseLib）
2. 前往本仓库的 [Releases](https://github.com/dreamyloong2005/SpireOfInfinity/releases) 页面，下载最新的模组文件。
3. 将下载的文件放入游戏根目录下的 `mods\SpireOfInfinity` 文件夹中。
    * *Windows 默认路径示例*: `\Steam\steamapps\common\SlayTheSpire2\mods`
4. 启动游戏，允许模组加载，重启游戏。

## ⚙️ 兼容性与前置要求 (Requirements & Compatibility)

* **游戏版本**：需要public-beta版本。
* **前置模组**：
    BaseLib v0.2.6
* **已知冲突**：暂无
